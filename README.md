## Installation

- Python
	1. Have python & pip installed
	2. "pip install -r requirements.txt"
	3. open ./nn_model/generate.ipynb with method of choice (e.g. VSCode + Jupyter extension)

- React Native
	1. Have npm installed
	2. "npm install"
	3. "expo start"
	4. Open on web or Expo Go (Android & iOS)
